üß† What is Metadata?

**Metadata** is "data about data." In HTML, it's information about your webpage that isn't shown directly but helps browsers and search engines understand it. Metadata goes inside the <head> section, usually with a <meta> tag.

üè∑Ô∏è Meta Tags: Simply Explained

Meta tags often look like this: <meta name="description" content="This is a page about dogs.">

They usually have two main parts:

  • **`name`**: Tells what kind of information you're providing (e.g., "description", "keywords", "author").
  • **`content`**: The actual value of that information.
---

‚úÖ Important Meta Tags (With Examples)

1Ô∏è‚É£ <meta charset="UTF-8">

+

This sets the **character encoding**. **UTF-8** supports almost all languages and symbols (like emojis, Hindi, English). Always include it to avoid strange characters!

Example: <meta charset="UTF-8">

2Ô∏è‚É£ <meta name="description" content="...">

+

A short **summary** of your page. Search engines (like Google) often use this text in search results.

Example: <meta name="description" content="A free HTML tutorial for beginners.">

3Ô∏è‚É£ <meta name="keywords" content="...">

+

A list of **important words** about your page. Less crucial for modern search engines but still sometimes seen.

Example: <meta name="keywords" content="HTML, Web design, Tutorial">

4Ô∏è‚É£ <meta name="author" content="...">

+

Simply states **who created** the webpage.

Example: <meta name="author" content="Divyanshu Kumar">

5Ô∏è‚É£ <meta name="viewport" content="...">

+

**Crucial for mobile devices!** This makes your website look good and adapt to different screen sizes (phones, tablets). Without it, your site might look tiny on a phone.

Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">

What it means: width=device-width uses the device's full screen width. initial-scale=1.0 sets the default zoom level.

6Ô∏è‚É£ <meta http-equiv="..."> (Advanced)

+

These are special meta tags that control browser behavior.

  • **Refresh page:** <meta http-equiv="refresh" content="5"> (reloads every 5 seconds).
  • **Redirect to another page:** <meta http-equiv="refresh" content="0; url=https://example.com"> (sends user to another site immediately).
