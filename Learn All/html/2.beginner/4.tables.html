üîç What is a Table in HTML?

Tables in HTML are used to arrange data in rows and columns, just like a spreadsheet or an Excel sheet.

You can use tables to show:

  • Schedules
  • Price charts
  • Student marks
  • Product lists
  • Contact details

üß± Basic Structure of a Table

<tr>
<th>Heading 1
<th>Heading 2
</tr>
<tr>
<td>Data 1
<td>Data 2
</tr>
</table>

üß† Easy Meaning of Tags:

Tag Stands For Meaning & Use
<table> Table Starts the table
<tr> Table Row Defines a row in the table
<th> Table Header Cell Defines a heading cell (often bold and centered by default)
<td> Table Data Cell Defines a normal data cell
<caption> Table Caption Gives a title to the table (optional)
colspan Column Span An attribute used in `<th>` or `<td>` to merge multiple columns into one cell
rowspan Row Span An attribute used in `<th>` or `<td>` to merge multiple rows into one cell

üß† Summary Table

Tag or Attribute Use For...
<table> Start of a table
<tr> A row in the table
<td> A regular data cell
<th> A header cell (bold and centered by default)
<caption> Title of the table
colspan="n" Attribute for `<th>` or `<td>` to make a cell stretch across `n` columns
rowspan="n" Attribute for `<th>` or `<td>` to make a cell stretch across `n` rows
border="1" An **outdated** attribute for `<table>` to add a visible border (use CSS `border` property instead for modern web development)
width="..." An **outdated** attribute for `<table>` to set the width of the table (use CSS `width` property instead)
align="..." An **outdated** attribute for `<table>` to align the table (use CSS `margin: auto;` for centering block elements, or `text-align` for inline content)
