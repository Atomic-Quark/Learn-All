JavaScript Mastery

The programming language of the web. Make things interactive.

What is JavaScript? (The Brain)

If HTML is the skeleton and CSS is the clothes, JavaScript is the brain.

It makes the website do things.

  • It handles clicks (Like the "Like" button).
  • It does math (Calculator).
  • It talks to servers (Loading new posts).

Your First Script

We use console.log to talk to the computer.

console.log("Hello World!");
// Prints "Hello World!" to the console

Variables & Data

Variables are like boxes where we store information.

let x = 5;

A number box.

let name = "John";

A text box (String).

const pi = 3.14;

A box that can't change.

Functions & Logic

Functions are little machines that do a specific job.

function greet(name) {
  return "Hello " + name;
}

greet("Alice"); // Returns "Hello Alice"

If / Else

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Kid");
}

DOM Manipulation

Changing the HTML using JavaScript.

const btn = document.getElementById("myBtn");
btn.addEventListener("click", () => {
  alert("Clicked!");
});

Async & API

Talking to the outside world (Servers, Databases).

async function getData() {
  const response = await fetch("https://api.com/data");
  const data = await response.json();
  console.log(data);
}
