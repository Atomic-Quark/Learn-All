CSS Mastery

Cascading Style Sheets - making the web beautiful.

What is CSS? (The Clothes)

If HTML is the skeleton, CSS is the skin and clothes.

Without CSS, every website would look like a boring Word document from 1995.

  • It adds color (Red shirt? Blue pants?).
  • It changes size (Big hat? Small shoes?).
  • It moves things around (Put the logo on the left).

Your First Style

CSS looks like a list of rules.

h1 {
  color: red;
  font-size: 50px;
}

Selectors & Colors

How do we tell CSS which element to style? We use Selectors.

p { ... }

Selects ALL paragraphs.

.classname { ... }

Selects elements with specific class.

#idname { ... }

Selects ONE unique element.

Box Model & Flexbox

Every element in HTML is a box. CSS controls that box.

The Box Model

  • Content: The text or image.
  • Padding: Space INSIDE the border.
  • Border: The line around the box.
  • Margin: Space OUTSIDE the border.

Flexbox

The magic tool to align things side-by-side easily.

.container {
  display: flex;
  justify-content: center;
}

Grid & Animations

CSS Grid is for 2D layouts (rows AND columns). Animations bring life.

@keyframes slide {
  from { left: 0; }
  to { left: 100px; }
}

Variables & Preprocessors

Modern CSS is like a programming language itself.

CSS Variables

:root {
  --main-color: #ff0000;
}
